
## Python Maze Generator

The python maze generator creates a maze from one point to another on a grid of cells
ensuring that every part of the maze is accessible with no loops.

![Demo gif](gifs/maze-generator-gif.gif)

## Features

- Randomised start and end.
- End is always located on the opposite side of the start.
- Animated maze drawing from start point.

## Installation

```sh
git clone https://github.com/stvnhu/maze-generator.git
```

- tkinter
